These folders contain all of the finalized code required to interface and operate the ADALM-PLUTO Rev C SDR.

This interface is intended to be performed with a Raspberry Pi 5B running Raspbian Lite 64-bit.

This Interfacing is performed over USB 2.0 (as is native for the ADALM-PLUTO Rev C).

This repository DOES NOT include code required to interface the LSM6DSO IMU, LIS3MDL Magnetometer, MS5607-02BA03 Altimeter, Digi XBee 3 Pro, nor Radio Frequency Front End boards with the Raspberry Pi 5B.

Code for interfacing the ADALM-PLUTO Rev C SDRs with the Raspberry Pi 5B can be found at:

[https://github.com/SL-UAV-MQP/pi_C_code2 
https://github.com/SL-UAV-MQP/pi_C_code](https://github.com/SL-UAV-MQP/Attitude_Sensor_Development)
